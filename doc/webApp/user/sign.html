个人 / 企业模块 - 注册
请求位置 user.sign
实现人 涂先锋
请求参数 参数 类型 必填项 说明
type int 必 1=个人注册 , 2=企业注册
phone bigint 必 手机号码
vxCode int 必 图像验证码
smsCode int 必 短信验证码
password_1 string 必 密码1
password_2 string 必 密码2
reCode int 推荐码
agree int 必 是否同意注册协议 , 1=同意
以下是企业注册 , 还需要传递的参数
com_name string 必 公司名称
one_id int 必 省 ID
two_id int 必 市 ID
three_id int 必 区/县 ID
com_address string 必 公司地址
com_num string 必 公司人数
com_property string 必 公司类型
com_license string 必 营业执照
com_license_timeout string 必 营业执照到期时间 (YYYY-MM-DD : 2012-12-12)
com_tax string 必 税务登记
com_org string 必 组织机构代码
正常的返回值 

{
	"code": 0,
	"x-token": "2319594324004b25f29ac0e74f9cf57a",
	"message": [],
	"data": []
}

	请注意
	
	1) 如果是注册个人会员 , 在返回值正常情况下 , 请跳转到个人中心

	2) 如果是注册企业会员 , 在返回值正常情况下 , 请跳转到待审核界面
