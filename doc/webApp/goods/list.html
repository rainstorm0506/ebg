商品模块 - 列表
请求位置 goods.list
实现人 谭甜
请求参数 参数 类型 必填项 说明
keyword string 选填 搜索关键字
o string 选填 商品排序字段(price(价格)，putaway（上架时间）,detail(销量))
by string 选填 商品排序方式(asc(升序)，desc(降序))
b int 选填 商品品牌id
ps decimal 选填 价格区间（开始价格）
pe decimal 选填 价格区间（结束价格）
self int 选填 自营（1=查看自营商品）
classOne int 选填 一级分类id
classTwo int 选填 二级分类id
classThree int 选填 三级分类id
pageNow int 必 页码（默认为第一页开始）
pageSize int 选填 每页显示的条数（默认为每页6条）
attr array 选填 商品属性
