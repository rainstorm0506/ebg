供应模块 - 搜索列表
请求位置 supply.search
实现人 何铭
请求参数 apt int APP抛数据的时间
page int 页码
title string 商品名称和商家
brand_id int 品牌ID
cid int 分类ID
type int 共需类型(供应=1;需求=2)
size_id int 尺寸ID
color_id int 颜色ID
order int 排序(时间倒叙=1;价格正序=2;价格倒叙=3)
